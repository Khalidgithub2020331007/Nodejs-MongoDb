# To Start MongoDB 
```
sudo systemctl start mongod
```
or 
```
sudo service mongod start
```
# To Check Status
```
sudo systemctl status mongod
```
or
```
sudo systemctl enable mongod
```
# To Check MongoDB Version
```
mongod --version
```
# To Stop MongoDB
```
sudo service mongod stop
```
# To Restart MongoDB
```
sudo service mongod restart
```
